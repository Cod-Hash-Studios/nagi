Nagi 目前尚未发布签名二进制文件或包管理器版本。继承的下载渠道已被禁用，
避免以 Nagi 的名义安装上游或未签名的二进制文件。

## 环境要求

- Rust 1.96.1
- Zig 0.15.2
- 任务运行时需要 Unix 系统

## 构建

```bash
git clone https://github.com/Cod-Hash-Studios/nagi.git
cd nagi

zig version  # 必须输出 0.15.2
cargo build --release --locked
./target/release/nagi
```

## 验证

```bash
./target/release/nagi --version
./target/release/nagi api schema
```

用于重要工作前，请阅读 README 中的
[当前状态](https://github.com/Cod-Hash-Studios/nagi#the-honest-status)。

在 Nagi 自有的发布流程产出签名文件之前，自动更新保持禁用。
