---
title: Windows 支持状态
description: Nagi 当前的 Windows 支持范围与安全试用方式。
---

Nagi 目前不分发原生 Windows 二进制文件。继承的预览安装器不属于 Nagi 的发布流程，
因此已被禁用。

现阶段请先阅读[源码构建要求](/zh-cn/docs/install/)，并在 Unix 主机或 WSL 中使用。
原生 Windows 构建仍处于实验阶段，不属于 V1 的正式支持范围。

可在[公开仓库](https://github.com/Cod-Hash-Studios/nagi/issues)跟踪进展。
本页不会下载任何未签名二进制文件。
