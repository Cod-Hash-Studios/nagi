---
title: Windows status
description: Current Windows support and the safe path for trying Nagi.
---

Nagi does not currently distribute a native Windows binary. The inherited
preview installer is disabled because it does not belong to Nagi's release
pipeline.

For now, use Nagi on a Unix host or inside WSL after reviewing the
[source-build requirements](/docs/install/). Native Windows builds remain
experimental and are not part of the supported V1 surface.

Track Windows work in the
[public repository](https://github.com/Cod-Hash-Studios/nagi/issues). No command
on this page downloads an unsigned binary.
