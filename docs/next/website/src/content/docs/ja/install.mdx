---
title: Nagi をソースからビルド
description: 署名済みバイナリが公開されるまで、Nagi を安全にビルドして確認する方法。
---

Nagi はまだ署名済みバイナリやパッケージマネージャー向けリリースを公開していません。
継承されたダウンロード経路は、別プロジェクトや未署名のバイナリを Nagi として
インストールしないよう無効化されています。

## 必要なもの

- Rust 1.96.1
- Zig 0.15.2
- ミッションランタイム用の Unix 環境

## ビルド

```bash
git clone https://github.com/Cod-Hash-Studios/nagi.git
cd nagi

zig version  # 0.15.2 であること
cargo build --release --locked
./target/release/nagi
```

## 確認

```bash
./target/release/nagi --version
./target/release/nagi api schema
```

重要な作業に使う前に、README の
[現在の状態](https://github.com/Cod-Hash-Studios/nagi#the-honest-status)を確認してください。

自動更新は、Nagi 自身のリリースパイプラインから署名済み成果物を公開するまで無効です。
