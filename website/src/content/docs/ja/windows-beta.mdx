---
title: Windows の対応状況
description: Nagi の現在の Windows 対応と安全な試し方。
---

Nagi は現在、ネイティブ Windows バイナリを配布していません。継承された
プレビューインストーラーは Nagi のリリースパイプラインに属さないため無効です。

当面は [ソースビルド要件](/ja/docs/install/) を確認し、Unix ホストまたは WSL で
利用してください。ネイティブ Windows ビルドは実験段階で、V1 のサポート対象外です。

進捗は [公開リポジトリ](https://github.com/Cod-Hash-Studios/nagi/issues) で追跡できます。
このページから未署名バイナリをダウンロードすることはありません。
